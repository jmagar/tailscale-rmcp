# tailscale-rmcp docs

This directory contains the maintained project docs for `tailscale-rmcp`, the
Rust RMCP server and `rtailscale` CLI for the Tailscale REST API.

The source tree is the final authority for behavior. These docs should mirror:

- one MCP tool named `tailscale`
- the `rtailscale` binary and npm launcher `tailscale-rmcp`
- MCP Streamable HTTP on port `40040`
- local stdio MCP via `rtailscale mcp`
- host app data in `~/.tailscale-mcp`
- Tailscale API access through `TAILSCALE_API_KEY`
- the `{ ok, data, error }` tool result contract

## Reference snapshots

`docs/references/` is generated by `scripts/refresh-docs.sh` and contains
upstream Tailscale and MCP reference snapshots. Some upstream MCP logging docs
mention syslog severity levels, and some upstream Tailscale pages mention local
syslog. Those are reference-source terms, not project behavior.
